Tracker.Sptfy

Utalise this to track your music listening habits on spotify. Currently, this python program requires you to set up a spotify for developers app. You can do so here.

Once done, set up a .env file with the following format:

SPOTIFY_CLIENT_ID=
SPOTIFY_CLIENT_SECRET=
SPOTIFY_REDIRECT_URI=http://localhost:8888/callback

Now running the program through command prompt will save you stats !
